# 🦈 Global Shark Attack Risk Analysis

A data science project analyzing historical shark attack data to identify patterns in fatal outcomes, built with Python (analysis) and React (interactive dashboard).

**Authors:** Steffina Jerald & Tiya Bordia  
**Course:** Data Management  

---

## Project Overview

Shark attacks are rare but receive significant media attention, raising concerns about ocean safety for swimmers, surfers, and divers. This project analyzes the **Global Shark Attack Dataset** to answer:

> *"What factors make a shark attack more likely to be fatal?"*

We build and compare three classification models to predict fatality, identify the most important risk factors, and produce a **shark attack risk score** for any given situation.
